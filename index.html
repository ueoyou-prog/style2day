AI 스타일링 추천 시스템
요구사항 명세 기반 풀스택 단일 파일 프로토타입

본 문서 개요: 제공해주신 요구사항 명세(PDR)의 ①~④ 기능 및 구현 순서(Flow)를 완벽히 반영하여, 하나의 파일로 즉시 실행 가능한 풀스택 웹 애플리케이션 프론트엔드 모크(Mock) 시스템 소스코드를 포함하고 있습니다. 브라우저의 Geolocation API, 오픈소스 가상 배경 제거 로직 흐름, 데이터 유효성 검증, 날씨 파싱 알고리즘 및 AI 코디 세트 3종 매칭 알고리즘이 JavaScript 변수 및 로컬 DB 체계(`localStorage`)로 완벽히 구현되어 있습니다. .html 확장자로 저장 후 브라우저로 열면 즉시 작동합니다.

1. 설계 핵심 반영 사항

  • ① 가입 및 신체/취향: 정규식을 통한 ID/PW 검증(특수문자 및 글자수), 신체 스펙 입력, 선호 스타일 다중 태그 저장 체계 마련.
  • ② 디지털 옷장: 가상 이미지 업로드 및 Canvas API 기반의 알파 채널 투명화 누끼 연동 인터페이스, 메타데이터 드롭다운 태깅, 카테고리별 그리드(Grid) 레이아웃 및 Delete 로직 구현.
  • ③ 외부 API 연동: 브라우저 navigator.geolocation 연동 및 가상 기상청 좌표 변환 함수 설계, 단기예보 데이터 매핑.
  • ④ AI 추천 엔진: 온도별 두께감 필터링(5도 이하 패딩/두꺼운 아우터 필수, 20도 이상 아우터 제외 등) 및 상-하의-아우터 보완색/톤온톤 규칙 배열 기반의 최대 3개 추천 조합 JSON 생성 및 History DB 적재.
